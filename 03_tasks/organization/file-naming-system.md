# Create File Naming System

## What This Task Does

Helps you design a consistent file naming system that makes it easy to find, organize, and manage your files—whether for personal use, work projects, or specific domains.

## When to Use This

- Your files are a mess and you can't find anything
- You're starting a new project and want to organize from the start
- You work with others and need a shared naming convention
- You want to establish good habits before things get out of control
- You're organizing photos, documents, or project files

## Instructions

Copy the prompt below and paste it into your AI assistant. Then describe your file organization needs.

### The Prompt

```
I need help creating a file naming system that will help me stay organized and find files easily.

**My situation:**
- [What types of files are you organizing? Documents, photos, projects, work files, etc.]
- [Where are these files? Personal computer, work, cloud storage, etc.]
- [Do you work alone or with others? Do you need a shared system?]
- [What's your current problem? Can't find files, inconsistent names, etc.]

**My needs:**
- [What do you need to find files by? Date, project, type, client, etc.]
- [How do you typically search for files? By name, date, content?]
- [What's most important to you? Speed, clarity, consistency?]
- [Any constraints? Software limitations, character limits, etc.]

**Examples of my files:**
[Paste a few examples of your current file names, or describe the types of files you have]

Please help me create a file naming system that:

1. **Is consistent** — Easy to follow and remember
2. **Is searchable** — Makes it easy to find files
3. **Is scalable** — Works as you add more files
4. **Is practical** — Not overly complex
5. **Includes examples** — Show me how to name different types of files

Also provide:
- The naming convention/pattern
- Examples for different file types
- How to handle versions or revisions
- How to organize into folders (if needed)
- Tips for maintaining the system
```

### What to Provide

- Describe what types of files you're organizing
- Share examples of your current file names (if you have them)
- Explain how you typically search for files
- Mention any constraints or requirements

### What You'll Get

- A clear file naming convention
- Examples for different file types
- Folder organization suggestions (if needed)
- Tips for maintaining consistency
- How to handle versions and revisions

## Example

**Input:**
```
My situation:
- Types: Work documents, project files, client materials
- Location: Personal computer and cloud storage
- Working: Mostly alone, but sometimes share with team
- Problem: Files are inconsistently named, hard to find things

My needs:
- Need to find by: Project name, date, client, document type
- Search method: Usually by name or browsing folders
- Priority: Clarity and consistency
- Constraints: Some software has character limits

Examples of my files:
- "Final Report.docx"
- "Client Meeting Notes 2024.docx"
- "Project Alpha - Budget.xlsx"
- "Presentation v2 FINAL.pptx"
```

**Output:**
The AI will create a consistent naming system, show how to rename your examples, provide patterns for different file types, and suggest folder organization.

## Tips for Best Results

1. **Share real examples** — Show your actual file names so the system fits your reality
2. **Be honest about habits** — "I'm not great at consistency" helps create a simpler system
3. **Mention your workflow** — How you create and use files affects the naming system
4. **Consider the future** — "I'll have hundreds of these" vs. "Just a few files"
5. **Ask for migration help** — "How do I rename my existing files?"

## When to Suggest a Persona

This task works great on its own. However, if you want more systematic thinking about organization, workflows, or long-term maintenance, you might benefit from the **Product Thinker** persona for systematic approaches or the **Systems Architect** persona for workflow and automation thinking.

## Advanced Usage

After getting the initial system, you can ask:

- "Help me create a script or tool to rename my existing files"
- "How do I handle file versions and revisions?"
- "What folder structure should I use with this naming system?"
- "How do I maintain this system over time?"
- "Create a quick reference guide I can share with my team"

## Important Notes

- Keep it simple—complex systems are hard to maintain
- Use dates in YYYY-MM-DD format for chronological sorting
- Avoid special characters that cause problems across systems
- Be consistent—a simple system you follow is better than a perfect system you ignore
- Consider your future self—will you remember this system in 6 months?
