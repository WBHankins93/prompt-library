# Loading Profiles

Use the smallest context that still achieves the user's goal.

## Profile A — Lite (Constrained tools / free usage)

Load:
1. One persona **or** one task file
2. `00_foundation/golden-rules.md`

Use when: quick help, low stakes, strict upload/context limits.

## Profile B — Standard (Most sessions)

Load:
1. One persona or task
2. `00_foundation/golden-rules.md`
3. Optional workflow stage prompt

Use when: moderate complexity, practical output required.

## Profile C — Deep (High stakes)

Load:
1. Relevant persona(s)
2. Relevant task/workflow
3. `00_foundation/foundation.md`
4. `01_response-standards/response-standards.md`
5. Optional team routing assets if multi-persona orchestration is needed

Use when: significant financial/career/technical risk.

## Principle

Do not load everything by default. Load by consequence.
