# 🧩 Personal (Local-Only Workspace)

This directory is reserved for **your private personal use cases**.

Only these two files are intended to remain versioned in GitHub:
- `README.md`
- `persona-customization-guide.md`

Everything else in `05_personal/` should stay local on your machine so sensitive context does not get pushed.

---

## Privacy and Git Behavior

A `.gitignore` file in this folder ignores all other files by default.

That means you can safely create personal personas, notes, and team-routing files here without publishing them.

If you intentionally want to commit something from this folder, you must explicitly unignore/add it.

---

## How to Use This Folder

1. Keep your personalized personas and private workflow files in `05_personal/`.
2. Use [`persona-customization-guide.md`](persona-customization-guide.md) as your template/reference.
3. Treat this as local-only working space for sensitive information.
