# Naming Quick Checklist

Use this before opening a PR that adds or renames files.

## 60-Second Pass

- [ ] Is every new filename lowercase kebab-case?
- [ ] Does each filename describe a concrete role or outcome?
- [ ] Any `-v2`, `-final`, or temporary naming?
- [ ] Any overlap with existing file names?
- [ ] If this is a rename, were all references updated?

## Escalate Review If

- The rename changes user-facing meaning.
- The rename affects multiple directories.
- The rename changes taxonomy (persona/task/workflow grouping).
