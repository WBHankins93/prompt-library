# 06 Internal Assets

Operational assets for maintaining and scaling the prompt library.

## Purpose

This directory contains non-runtime assets used by maintainers:
- governance and policies
- quality checks and evaluation specs
- schema definitions
- release procedures

## Subdirectories

- `docs/` — governance and contribution documentation
- `quality/` — evaluation plans, golden tests, and QA assets
- `schemas/` — format schemas for personas/tasks/workflows
- `release/` — release and checkpoint procedures

Runtime prompt content remains in `00`-`05`.
